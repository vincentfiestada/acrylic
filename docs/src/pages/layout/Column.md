## Column

A **Column** lays out its children in a single column.

```jsx
render(() => (
	<Column>
		<Card>
			<CardImage src="/img/art1.webp" />
		</Card>
		<Card>
			<CardImage src="/img/art2.webp" />
		</Card>
		<Card>
			<CardImage src="/img/art3.webp" />
		</Card>
	</Column>
))
```
