## Row

A **Row** lays out its children in a single row.

```jsx
render(() => (
	<Row>
		<Card classes="medium">
			<Image src="/img/art1.webp" />
		</Card>
		<Card classes="medium">
			<Image src="/img/art2.webp" />
		</Card>
		<Card classes="medium">
			<Image src="/img/art3.webp" />
		</Card>
	</Row>
))
```
