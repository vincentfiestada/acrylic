## Nav

A **Nav** is a semantic container for navigation.

```jsx
render(() => (
	<Nav elevation={1} classes="medium">
		<Menu>
			<MenuItem><a href="#">File</a></MenuItem>
			<MenuItem><a href="#">Edit</a></MenuItem>
			<MenuItem><a href="#">View</a></MenuItem>
			<MenuItem><a href="#">Help</a></MenuItem>
		</Menu>
	</Nav>
))
```
