## Loader

A **Loader** represents an indeterminate task currently in progress. It supports the standard variants.

```jsx
render(() => (
	<>
		<Loader />
		<Loader variant="primary" />
		<Loader variant="success" />
		<Loader variant="warning" />
		<Loader variant="danger" />
	</>
))
```
