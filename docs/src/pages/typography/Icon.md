## Icon

An **Icon** displays an image that scales with the current font size.

```jsx
render(() => (
	<div>
		<Text>
			<Icon src="/src/img/ok.svg" />
			<Icon src="/src/img/info.svg" />
			<Icon src="/src/img/cancel.svg" />
			<Icon src="/src/img/trash.svg" />
		</Text>
		<Title>
			<Icon src="/src/img/ok.svg" />
			<Icon src="/src/img/info.svg" />
			<Icon src="/src/img/cancel.svg" />
			<Icon src="/src/img/trash.svg" />
		</Title>
	</div>
))
```
