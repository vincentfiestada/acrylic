## Icon

An **Icon** displays an image that scales with the current font size.

```jsx
render(() => (
	<div>
		<Text>
			<Icon src="/img/ok.svg" />
			<Icon src="/img/info.svg" />
			<Icon src="/img/cancel.svg" />
			<Icon src="/img/trash.svg" />
		</Text>
		<Title>
			<Icon src="/img/ok.svg" />
			<Icon src="/img/info.svg" />
			<Icon src="/img/cancel.svg" />
			<Icon src="/img/trash.svg" />
		</Title>
	</div>
))
```
