## Development

The library and the docs can be built using the [Parcel](https://parceljs.org/) bundler.

```sh
yarn install
yarn format
yarn build:dev
```

Tests are written using [Jest](https://jestjs.io/) and can be run like so:

```sh
yarn test
```
